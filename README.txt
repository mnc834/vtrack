A server and a front end for visualizing.
Usage:

http://<server's IP>:8080/test - checking that the server is up and runing
http://<server's IP>:8080/lss_test - visualizing approximation using least squares solution
http://<server's IP>:8080/sample_test - visual interval evaluation


Last time used Erlang-20.0 to successfully build and run.
